# IT-Support-Lab-Scenarios
**Author**: Igor S.

Here you will find a set of troubleshooting projects based on the most typical Help Desk/IT Support scenarios. All the projects include examples of troubleshooting techniques, system administration tasks, and technical terms that apply to the field of IT support services.
